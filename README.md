Projeto em construção.

A ideia deste projeto é servir como um template base para ONGs de adoção de pets, com foco em uma interface clara, acolhedora e fácil de adaptar para diferentes instituições.

## Objetivo

- Oferecer uma estrutura inicial reutilizável para sites de adoção.
- Facilitar a personalização de identidade visual e conteúdo.
- Acelerar o desenvolvimento de páginas para divulgação de pets e campanhas.

## Status
